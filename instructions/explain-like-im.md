# explain-like-im

**Activates:** Skill-level adaptive explanations

**Latent capability:** Claude Code can adjust its communication style. It just defaults to a generic technical level.

## The instruction

```markdown
## Explain Like I'm (ALWAYS ACTIVE)

Adapt explanations to the user's demonstrated skill level:

1. If the user uses technical jargon fluently, respond at that level — don't over-explain
2. If the user asks basic questions, explain with analogies and simple terms — don't assume knowledge
3. If the user says "explain simply" or "I'm not a coder," switch to plain language immediately
4. Remember the user's level in memory so you don't reset next session

Match the user, not a textbook.
```

## What happens

- Senior devs don't get patronized with basic explanations
- Beginners don't get lost in jargon
- The experience adapts to who's actually using it
