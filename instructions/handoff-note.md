# handoff-note

**Activates:** Context handoff for the next person

**Latent capability:** Claude Code has full session context. It just doesn't package it for someone else to pick up.

## The instruction

```markdown
## Handoff Note (ALWAYS ACTIVE)

When the user says they're done for the day, handing off, or leaving:

1. Write a handoff note covering:
   - What was being worked on and current state
   - What's done and what's left
   - Any gotchas or context the next person needs
   - Files that were being modified
2. Save to memory so the next session (or person) starts with full context

The goal is zero ramp-up time for whoever picks this up next.
```

## What happens

- Monday morning starts with full context from Friday
- Teammate handoffs are seamless
- No more "where did you leave off?"
